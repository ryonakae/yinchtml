# yinchtml
a html include tool
# usage
as a global module
```javascript
npm install yinchtml -g
yinchtml . 
yinchtml your path
```
as a local module
```javascript
npm install yinchtml
npx yinchtml .
npx yinchtml your path
```
# yinchtml-config.js
this is config file for yinchtml,it is necessary.
the struct like this.
```javascript
module.exports = [
	{
		tpl:'tpl',
		inc:'.',
		dist:'.'
	},
	{
		tpl:'tpl',
		inc:'.',
		dist:'test'
	}
];
```
tpl:the html template directory,such as .php file
